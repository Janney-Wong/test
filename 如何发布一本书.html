Test1

  1. 注册并登陆GitHub账号https://github.com/
  2. 创建一个名为 Github用户名.github.io 的仓库（repository）
  3. 访问https://janney-wong.github.io/ 报404
  4. 在本地找个位置,用命令安装gitbook npm install -g gitbook -cli
  5. 创建文件夹 mkdir test cd test
  6. gitbook init
  7. gitbook serve -- 起本地服务
  8. 完成的项目上传到github上以后，默认的是处于master分支 git init git remote add origin https://github.com/Janney-Wong/test.git git add . git commit -m 'init' git push -u origin master
  9. 我们要做的是展现dist目录下的静态文件，最关键的语句 git subtree push --prefix=_book origin gh-pages
  10. 然后别人就可以访问你的项目啦
